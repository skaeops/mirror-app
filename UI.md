## 1. 视觉风格

### 1.1 设计理念

> **"让照片成为主角"**

- 极简、克制、退后
- UI 是画框,不是画
- 致敬经典摄影美学

### 1.2 关键词
- **Minimal** — 少即是多
- **Cinematic** — 电影感
- **Dark** — 深色优先
- **Tactile** — 有触感的交互

### 1.3 参考应用

| App | 参考点 |
|-----|--------|
| **Glass** | 纯净的摄影社区审美、深色模式 |
| **VSCO** | 滤镜 UI、全屏浏览 |
| **Darkroom** | 专业感、手势交互 |
| **Apple Photos** | 原生 iOS 体验、动效 |
| **Halide** | 相机取景器美学 |

## 2. 色彩系统

### 2.1 主色板

```
背景层级:
├── Background Primary:    #000000 (纯黑)
├── Background Secondary:  #0A0A0A (微灰)
├── Background Tertiary:   #1A1A1A (卡片)
└── Background Elevated:   #2A2A2A (弹窗)

文字层级:
├── Text Primary:          #FFFFFF (标题)
├── Text Secondary:        #A0A0A0 (正文)
├── Text Tertiary:         #666666 (辅助)
└── Text Disabled:         #404040 (禁用)

强调色:
├── Accent Primary:        #F5D547 (黄斑色 - 徕卡黄)
├── Accent Secondary:      #E8C33A (深黄)
└── Accent Subtle:         #3D3520 (暗黄背景)

功能色:
├── My Work:               #4A9EFF (蓝色标记)
├── Inspiration:           #FF6B6B (红色标记)
├── Similar High:          #4ADE80 (高相似度)
└── Similar Medium:        #FBBF24 (中相似度)
```

### 2.2 颜色使用原则

- **黑色为王** — 90% 的界面是黑色
- **颜色点睛** — 只在关键位置用颜色
- **徕卡黄** — 品牌色,用于导入动效、重要操作
- **无彩色优先** — 不与照片颜色冲突

## 3. 字体系统

### 3.1 字体选择

| 用途 | 字体 | 理由 |
|------|------|------|
| **中文** | SF Pro Text | 系统默认,清晰 |

```
├── Title 3:    17pt  Semibold (卡片标题)
├── Body:       17pt  Regular  (正文)
├── Callout:    16pt  Regular  (次要文字)
├── Subhead:    15pt  Regular  (辅助信息)
├── Footnote:   13pt  Regular  (小字)
└── Caption:    11pt  Regular  (标注)
```

## 4. 布局系统

### 4.1 间距

```
├── xs:  4pt   (紧凑间距)
├── sm:  8pt   (小间距)
├── md:  16pt  (标准间距)
├── lg:  24pt  (大间距)
├── xl:  32pt  (区块间距)
└── xxl: 48pt  (页面间距)
```

### 4.2 瀑布流

- 双列布局
- 列间距: 4pt (极小,照片几乎挨着)
- 行间距: 4pt
- 边距: 0pt (全屏出血)

### 4.3 全屏查看

- 无边框
- 照片占满屏幕
- Safe Area 透明覆盖
- 手势优先,无按钮

## 6. 动效设计

### 6.1 核心动效

| 场景 | 动效 | 时长 | 曲线 |
|------|------|------|------|
| 黄斑对焦 | 双影像合一 | 0.8s | easeInOut |
| 照片切换 | 垂直滑动 | 0.3s | easeOut |
| 进入全屏 | 缩放展开 | 0.35s | spring |
| 退出全屏 | 缩小回位 | 0.3s | easeIn |
| 显示关联 | 左滑滑出 | 0.3s | easeOut |
| 加载照片 | 淡入 | 0.2s | linear |

### 6.2 触感反馈

| 场景 | 反馈类型 |
|------|---------|
| 照片导入成功 | .success |
| 发现关联 | .light |
| 切换照片 | .selection |
| 长按操作 | .medium |

## 7. 图标设计

### 7.1 App 图标概念

```
方案 A: 双圆重叠 (黄斑对焦)
┌─────────────┐
│    ◐◑       │  两个圆形部分重叠
│             │  代表对焦/镜像
│             │  徕卡黄 + 白色
└─────────────┘

方案 B: M 镜像
┌─────────────┐
│    M|M      │  M 和其镜像
│             │  代表 Mirror
│             │  简洁几何
└─────────────┘

方案 C: 取景器框
┌─────────────┐
│   [   ]     │  取景器框
│     ⊙       │  中心对焦点
│             │  摄影感
└─────────────┘
```

### 7.2 功能图标

使用 SF Symbols,保持一致性:
- 首页: `house.fill`
- 发现: `sparkles`
- 添加: `plus.circle.fill`
- 我的作品: `camera.fill`
- 灵感: `star.fill`
- 设置: `gearshape.fill`
- 关联: `arrow.left.arrow.right`
